Database: is an organized collection of data stored electronically

Database Components:
1. Database: The full collection of data
2. Tables: A group of related data ex: students
3. Row/Record: One entry in the table ex: student_1
4. Column/Field: A property of the data  ex: name/email/ID
5. Primary Key (PK): Unique identifier per row
6. Foreign Key (FK): Links two tables.  ex: student_id in grades table

Why use databases:
1. Fast data retrieval
2. Consistnet structure
3. Secure storage





SQL Categories:
(Data Definition Language)   DDL   (CREATE, ALTER, DROP) -> Table level
(Data Manipulation Language) DML   (INSERT, UPDATE, DELETE) -> Row level
(Data Query Language)        DQL   (SELECT)
(Data Control Language)      DCL   (GRANT, REVOKE)
